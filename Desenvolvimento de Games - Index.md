# Desenvolvimento de Games — Disciplina

## Evolução do Projeto

**v1 — O Herói dos Blocos**
Jogo de plataforma 2D em pixel art com herói chamado **Kael**, transportado para dentro do servidor AusTV. Missão: coletar 4 Fragmentos do Núcleo corrompido e salvar o mundo antes do colapso. Gênero: Plataforma 2D com elementos Metroidvania.

**v2 — Henesys: O Despertar dos Wisps**
Reboot com personagem **Rael**, ambientado agora no universo real do AusTV (Henesys). Incorpora sistemas do servidor: mcMMO, Wisps com raridades, Battle Pass, Leilões Sombrios. Gênero: RPG de Plataforma 2D.

**v3 — Henesys: O Despertar dos Wisps (versão final)**
Versão refinada da v2 com líderes de clã reais do AusTV (Blocão, Shinigami, Marcus, Ozielux), Besta (crossbow) como arma principal, mecânica de Sneak/Sprint inspirada no Minecraft, e narrativa emocional aprofundada sobre o mentor Power.

---

## Elementos Recorrentes

- **Jornada do Herói** (9 etapas de Campbell) — estrutura narrativa base de todos os GDDs
- **4 zonas temáticas** — cada zona tem dungeon, mini-boss e fragmento de progressão
- **Cronômetro de 72h in-game** — pressão temporal sem punir exploração
- **Hub Central neutro** — área segura entre as zonas com NPCs e upgrades
- **Vilão corrompido** — Vórtex (v1) / Titã de Obsidiana (v2-v3) com batalha em 3 fases

---

## Conexão com o AusTV Real

Os GDDs v2 e v3 são **cartas de amor ao servidor AusTV**, incorporando:
- Zonas baseadas nas regiões reais de Henesys (Caipira, Espacial, Pizzaria, Halloween)
- Clãs com líderes reais (Blocão, Shinigami, Marcus, Ozielux)
- Itens do resource pack real (skins de Besta, Espada, cosméticos)
- Plugins reais como base de mecânicas (mcMMO, MythicMobs, Battle Pass, Leilões Sombrios)
- Power como o fundador/mentor — personagem baseado na administração real do server
